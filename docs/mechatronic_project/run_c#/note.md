# RUN C# AND MATLAB
1. Connect PLC 
2. Turn ON Servo
3. Turn ON MATLAB
4. Stop Simulink and Start again
5. Typing in MATLAB Command: run_realtime
6. Check COM number and choose a COM pair (set in run_real_time.m) (Don't open COM before run_realtime)
7. run_realtime
8. Open COM in C# (965200)
9.  Enjoy !!!
